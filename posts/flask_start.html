Starting a Flask App

Step by step instructions to start your Flask app.
Dec, 2020

Making the Virtual Environment in Terminal:

$ python3 -m venv my_environment
$ source my_environment/bin/activate
$ pip3 install flask

Once the environment is created, create python file, and save it as app.py. Your first few lines of code should be:

from flask import Flask


app = Flask(name)


@app.route("/")
def home():
    return "Hello, Flask!"

Once the file is saved, and you have added HTML/CSS if desired, run:

$ export FLASK_APP=app.py
$ flask run
