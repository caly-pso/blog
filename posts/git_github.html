Git and GitHub Tricks

Dec, 2020

Creating a requirements document:

Navigate to your project folder, activate your virtual environment, then run the following command:

$ cd my/project/folder 
$ activate project_env
$ pip freeze > requirements.txt

Running a requirements document:

In your terminal:

$ cd my/project/folder/with/requirements  
$ activate project_env
$ pip install -r requirements.txt
